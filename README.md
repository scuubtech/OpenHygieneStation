# OpenHygieneStation

OpenSCAD 2021.01 parametric hygiene station concepts for the Autodesk Designathon brief.

## Status

This repository is being initialized as a reusable OpenSCAD engineering kit.

## Planned structure

- `common/` shared parameters, utilities, and render helpers
- `hardware/` extrusions, brackets, fasteners, hinges, handles, casters
- `plumbing/` pipes, elbows, valves, manifolds, faucets, drains
- `water/` carboys, bottles, tanks, bladder components
- `sinks/` sink primitives and basin modules
- `filtration/` visual filtration modules
- `structure/` frames, shells, panels, supports
- `poc1/` through `poc5/` five distinct concept assemblies

## OpenSCAD target

Target version: OpenSCAD 2021.01

## Naming

Modules use PascalCase and are written to be reusable across the five concepts.
